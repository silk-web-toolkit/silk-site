Documentation Pipelines

Everything in Silk begins with pipelines. Out of the box Silk is configured to use the 'ViewDrivenPipeline' which constructs a site out of a directory structure containing html templates and views.

Pipelines are composed of:

  • Generators - for retrieving source content
  • Transformers - for transforming source content
  • Serialisers - for writing content out in a desired format
