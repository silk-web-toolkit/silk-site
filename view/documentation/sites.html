Documentation Sites

Silk builds sites out of templates, views and components at compile time. Sites are expected to have a particular structure, this is listed below.

components
local components live here, these are specific only to this website
meta
top level items like 'robots.txt' or HTML5 cache manifests
resource
images, stylesheets and javascript
template
top level templates for defining consistent look and feel across pages
view
each page of our website
