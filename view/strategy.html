0.1.0 - Static Silk
Objective
Build a working draft for a CLI.
Goal
Productivity through intelligent design.
Success Criteria
Minimum number of steps to achieve domain tasks.
Principle of least surprise, users intuitively know what to do.
Enable cross platform (OS) use.
Responsive on a 'compile' time level.
Implementation Plan
Identify requirements by liaising with the community.
Document the specification for the CLI.
Build a prototype.
Measure productivity as compared to the earlier prototype.
