Getting started

Information: Silk is selfishly geared to Linux/OSX right now, although it does work on Windows there is no nice wrapper script and some of the functionality is not ported.

Get Silk

Get current version

Install

>./silk_install.sh

Update

>silk update

Create a new directory for your site

>mkdir mysite; cd mysite

See which site prototypes are available

>silk sites

Clone a site prototype

>silk site-clone some-site-prototype

Spin your site

>silk spin

View your site (OSX)

>open site/index.html
