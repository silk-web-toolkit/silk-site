Getting started See the screencast

Get Silk

Get current version

Install (Linux/OSX)

>./silk_install.sh

Install (Windows)

Double click on setup.exe

Update

>silk update

See which site prototypes are available

>silk sites

Clone a site prototype

>silk clone-site some-site-prototype optional-directory-name

Spin your site

>cd optional-directory-name
>silk spin

View your site (OSX)

>open site/index.html
